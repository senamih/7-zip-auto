# 7-Zip-Auto

圧縮ファイルを **ダブルクリック／ドラッグ＆ドロップ／関連付け** で受け取り、設定済みの 7-Zip（`7zG.exe`）で自動展開し、展開先フォルダをそのままファイラで開く Windows 用ユーティリティです。展開の進捗を一覧で確認でき、複数ファイルを一度に渡してもキューで順に処理します。

- 圧縮ファイルをいちいち展開するのが面倒な方におすすめです！
- 7-Zipが無い環境でもwingetできる環境なら7-Zipのインストールもやります！多分
- 設定画面で左上の設定を全部ONにするのがおすすめです！裏で展開だけをこなすマシンになります！
- Claude Codeオンリーでコードに一切触れていないのできっと安心！

## 動作環境

- Windows 10 / 11（64bit）
- [7-Zip](https://www.7-zip.org/) がインストール済みであること（本体は同梱しません。別途インストールしてください）
- .NET ランタイムのインストールは **不要**（単一 EXE に同梱した自己完結型）

## インストール

1. `7-Zip-Auto.exe` を任意のフォルダに置きます（配布物の `LICENSE` / `THIRD-PARTY-NOTICES.txt` / `README.md` / `CHANGELOG.md` も同じ場所に置くことを推奨）。
2. 初回起動時、`.7z` の関連付け → レジストリ → 既定インストール先（`C:\Program Files\7-Zip\7zG.exe` 等）の順で `7zG.exe` を自動検出します。見つからない場合は案内ダイアログから導入できます（後述）。

> インストーラはありません。フォルダごとコピーすれば移動・配布できます。設定は実行ファイルと同じ場所の `settings.json` に保存されます。

## 使い方

| 方法 | 動作 |
|---|---|
| 圧縮ファイルを EXE にドラッグ＆ドロップ | その場で展開 |
| 圧縮ファイルを右クリック →「プログラムから開く」で本アプリ | その場で展開（設定画面で拡張子を関連付け可能） |
| ウィンドウへドラッグ＆ドロップ | 一覧に追加して展開 |
| 引数にファイルパスを渡して起動 | 展開（複数指定可） |

- 展開先は、アーカイブと同じ階層の「**アーカイブ名（拡張子なし）**」フォルダです。アーカイブの中身が同名フォルダ 1 つだけの場合は二重フォルダを作りません。
- 既に同じアプリが起動している場合、新しく渡したファイルは起動中のウィンドウのキューへ追加されます（多重起動しません）。
- 一度に 2 件以上を渡すと一覧ウィンドウが自動表示され、順番に展開します。

## 対応書式

7-Zip 本体が対応する書式に準じます。設定画面で関連付け可能な拡張子は次のとおりです：

`.7z` `.zip` `.rar` `.tar` `.gz` `.bz2` `.xz` `.cab` `.iso` `.lzh` `.arj` `.wim` `.tgz` `.tbz`

> 関連付けの有無に関わらず、上記以外でも 7-Zip が展開できる書式であればドラッグ＆ドロップや引数指定で処理できます。

## タスクトレイ常駐・無人動作

- **常駐 ON**（設定）：ウィンドウを × で閉じても、一覧が空になってもプロセスは終了しません。タスクトレイアイコンを**左クリックで再表示**、**右クリックで「表示／設定／終了」**メニュー。終了はこのメニューの「終了」からのみ。
- **常駐 OFF**（既定）：ウィンドウを閉じる／一覧が空になるとプロセス終了。
- **無人動作**：常駐 OFF かつウィンドウ非表示の構成で引数からファイルを渡すと、画面を出さずに展開し、完了後フォルダを開いて自動終了します（バッチ処理向け）。

## 一覧ウィンドウ

各行に「ファイル名 / サイズ（圧縮・展開後）/ エントリ数 / 状態」と、2 つのアイコンボタンが並びます。

- 📂 **フォルダを開く**：展開先を設定したファイラで開きます。
- ✕ **削除**：その行を一覧から外します（実行中の展開処理はそのまま継続します）。
- メニューの「リストをクリア」で一覧を一括削除できます。
- 展開完了時は自動で展開先フォルダを開きます（設定で挙動を変更可）。

## 設定画面

メニューの「設定」から開きます。主な項目：

- **タスクトレイに常駐する**（既定 OFF）：ON でウィンドウを閉じても常駐し続け、トレイアイコンの左クリックで再表示。終了はトレイメニューの「終了」から。
- **一覧が空になったらウィンドウを自動で閉じる**（既定 OFF）
- **展開完了時に自動で一覧から削除する**（既定 OFF）
  - サブ：複数ファイルが渡された時は自動で削除しない（既定 OFF）
- **「フォルダを開く」ボタン押下時に自動で一覧から削除する**（既定 OFF）
- **7zG.exe のパス**：参照／検出ボタン付き。「検出」で自動検出を再実行できます。
- **開くファイラ**：エクスプローラ、または任意のファイラ（パス指定）。
- **拡張子の関連付け**：`.7z .zip .rar .tar .gz .bz2 .xz .cab .iso .lzh .arj .wim .tgz .tbz` をチェックで関連付け／解除（管理者権限不要・現在のユーザーのみ）。Windows 10/11 では「プログラムから開く」候補に登録され、既定アプリ化は Windows の設定から行ってください。

## トラブルシュート

- **起動しない／すぐ終わる**：圧縮単一 EXE は初回実行時に一時フォルダ（`%TEMP%`）へ展開して動作します。`%TEMP%` の空き容量不足や、ウイルス対策ソフトによる検疫が原因になり得ます。`%TEMP%` を空け、必要なら EXE を除外設定に追加してください。
- **7-Zip が見つからない**：7-Zip 未インストール時はガイド画面が表示されます。「7-Zip 公式サイトを開く」または「winget でインストール」で導入し、「インストール済み → 再検出」を押せばそのまま展開が続行されます（投入済みファイルは破棄されず待機中のまま残ります）。
- **展開されない**：設定画面で `7zG.exe` のパスが正しいか確認してください（「検出」ボタンで再検出できます）。
- **動作ログ**：実行ファイルと同じ場所の `7-Zip-Auto.log` に起動・展開・エラー内容が追記されます。不具合報告時に添付すると原因特定が早まります。

## 設定ファイル・ログ・アンインストール

| 種別 | 場所 | 説明 |
|---|---|---|
| 設定 | 実行ファイルと同じ場所の `settings.json` | 設定値とウィンドウ位置・サイズ。削除すると初期状態に戻ります。 |
| ログ | 実行ファイルと同じ場所の `7-Zip-Auto.log` | 起動・展開・エラーを追記。不具合報告時に添付すると診断が早まります。 |
| 関連付け | レジストリ `HKCU\Software\Classes`（現在のユーザーのみ） | 設定画面のチェックを外す（または下記）と解除されます。 |

**アンインストール**：本アプリはインストーラを使いません。

1. 設定画面で関連付けのチェックをすべて外して OK（レジストリの関連付けを解除）。
2. アプリを終了し、フォルダごと（`7-Zip-Auto.exe` / `settings.json` / `7-Zip-Auto.log` 等）削除。

> 7-Zip 本体は別ソフトのため、本アプリを削除しても 7-Zip は残ります（不要なら 7-Zip 側でアンインストールしてください）。

## よくある質問

- **7-Zip をインストールしていないと使えませんか？** はい。本アプリは展開を 7-Zip（`7zG.exe`）に委ねるため、別途インストールが必要です。未検出時は案内ダイアログから公式サイト／winget で導入できます。
- **管理者権限は必要ですか？** いいえ。関連付けは現在のユーザー範囲（`HKCU`）のみで、管理者権限は不要です。
- **既定アプリにできますか？** 「プログラムから開く」候補には登録されます。常に本アプリで開く設定は Windows の「既定のアプリ」から行ってください。
- **展開中にウィンドウを閉じても大丈夫？** 展開処理（7zG）は独立プロセスのため継続します。一覧から行を消しても展開自体は中断されません。
- **設定をやり直したい**：`settings.json` を削除して再起動すると初期状態（＋初回検出）からやり直せます。
